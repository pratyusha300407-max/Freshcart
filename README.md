🌾 FreshKart – Direct Farmer to Customer Marketplace

 📌 Project Description
FreshKart is a web-based platform that connects **farmers directly with customers**. The main goal of this project is to eliminate middlemen from the traditional agricultural supply chain and create a digital marketplace where farmers can sell their products directly to customers.

In the current agricultural system, farmers often depend on intermediaries to sell their produce. Because of this, farmers receive lower profits while customers pay higher prices. FreshKart solves this issue by providing a transparent platform where farmers can list their products and customers can buy them directly.

This platform benefits both farmers and customers by ensuring **fair pricing, fresh produce, and better communication between producers and consumers.**

---

 🎯 Objectives
- To create a **direct marketplace between farmers and customers**
- To eliminate **middlemen in agricultural sales**
- To help farmers **earn better profits**
- To allow customers to **buy fresh farm products directly**
- To provide a **simple and user-friendly online platform**

---

 👥 Target Users
- Farmers who want to sell their products directly
- Customers who want to purchase fresh agricultural products
- Admin who manages and monitors the system

---

⚙️ Features

👨‍🌾 Farmer Features
- Farmer registration and login
- Add new products with **name, price, quantity, and image**
- Edit or delete products
- View orders placed by customers
- Deliver products to customers
- Farmer login only after **admin approval**

### 🛒 Customer Features
- Customer registration and login
- View all available products in the marketplace
- Add products to cart
- Order products with selected quantity
- View cart and order details

### 👨‍💼 Admin Features
- Separate admin login
- Approve or reject farmer registration requests
- View total number of farmers and customers
- View delivery details
- Monitor the entire web application
- Track number of items sold

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository
```

```

### 2️⃣ Navigate to the Project Folder
```

```

### 3️⃣ Install Required Libraries
```

```

